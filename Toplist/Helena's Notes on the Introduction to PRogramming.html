I. The World Wide Web

The World Wide Web is a collection of HTML documents. HTML is the acronym for "Hypertext Markup Language". There are millions of HTML documents currently in the web. However, all types of files can be found on the web, including video, images, etc.

There are four main components of the web process:
Browser- A program that runs on the computer to display files from the web.Some examples include Internet Explorer, Firefox, and Chrome.
Internet- The world's largest computer network.
Servers- Computers that host the files that make up the web.
HTTP- The main protocal of the web.

II. HTML, Tags, and Elements

HTML (Hypertext Markup Language) is made up of the following:
Text Contet- What you see
Markup- What it looks like
References to Other Documents- images, vieods
Links to Other Pages

The standard exmple of an HTML Markup contains an opening and closing tag, with contents located within. This is usually referred to as an element.
"Opening Tag" --> Contents --> "Closing Tag"

HTML elements are categorized as either Inline or Block . The main difference between them is that block elements apply "boxes" around the content (text) while also factoring in height and width.

III. Structure of an HTML Document

The HTMLs documents are usually structured in the following way:

Section 1- The Head, which will contain the title of the document
Section 2- The Body, which will contain the main contents of the document

IV. Creating Your Own Webpage

Creating a webpage is similar to building a house. You have the walls and rooms as the building blocks. The HTML makes up the accents of the home, whether it's the color of the carpets or the type of furniture you put in. Java Script is the interactive portion of the home, which could be the garage openers, the electronics, etc. In the making of the webpage, users usually work with HTML, whcih is the standard markup language used to create the pages; and CSS, which is the cascading style sheets used to describe the look and formatting of the documents.
